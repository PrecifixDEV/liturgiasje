# App Skills & Rules

## Versionamento
- Sempre que houver uma alteração significativa no código ou uma nova funcionalidade for implementada, o número da versão exibido no rodapé da `page.tsx` deve ser incrementado em 0.01 (ex: de 1.03 para 1.04).
- Este número serve como indicador visual para o usuário de que o PWA foi atualizado com sucesso.

## Qualidade de Código
- Manter o padrão de Clean Code.
- Priorizar estética premium com CSS Vanilla/Tailwind consistente (variáveis do `globals.css`).
